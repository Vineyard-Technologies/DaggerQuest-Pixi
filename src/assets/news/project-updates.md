We're excited to share the latest developments for **DaggerQuest**!

## Migration to Pixi

We're porting **DaggerQuest** off of [Construct 3](https.Construct.net), and on to [Pixi v8](https://PixiJS.com). The reason behind this transition is primarily licensing costs. Be on the lookout for the 0.5.0 release of **DaggerQuest**. This will be the first Pixi release and should have feature parity with the [0.4.6](/news/patch-notes-046) Construct release.

## Art Direction Change

The decorative menu statues and full-body portraits will be removed in the Pixi version of **DaggerQuest**. We'll also be making new loading and menu screens. A core component of the **DaggerQuest** tech stack is [Daz Studio](https://www.daz3d.com/), which unfortunately is predominantly used in the creation of WEGs (Western Erotic Games). This is a necessary step in order to distance **DaggerQuest** from that genre.
